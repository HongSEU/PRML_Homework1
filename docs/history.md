# 项目历史

## 版本 0.1.0 (2026-03-28)

### 初始版本

- 实现Sonar数据集分类任务
- 支持Fisher's LDA和SVM两种算法
- 包含完整的数据加载、预处理、训练、评估流程
- 提供三种可视化图表：混淆矩阵、LDA分布、SVM决策边界
- 项目结构按照标准Python包规范组织

### 功能特性

- **数据加载**: 支持从UCI官方链接自动下载或本地文件读取
- **数据预处理**: 标签编码、数据划分、标准化处理
- **模型训练**: 
  - LDA: 线性判别分析，降维至1维
  - SVM: 支持向量机，带GridSearchCV超参数调优
- **模型评估**: 准确率、精确率、召回率、F1分数
- **可视化**: 高分辨率图表（300dpi），支持PNG和PDF格式

### 技术栈

- Python 3.8+
- NumPy >= 1.20.0
- Pandas >= 1.3.0
- Scikit-learn >= 0.24.0
- Matplotlib >= 3.3.0
- Seaborn >= 0.11.0
